# Fighter Jet Engine Cockpit Simulator (MFD F-26)

Node.js と Electron をベースにした、戦闘機用アフターバーナー付きターボファンエンジンのリアルタイム物理シミュレーターです。
実機のコックピットに搭載されている MFD（Multi-Function Display：多機能ディスプレイ）をイメージした、SFライクなネオングリーンのグラフィカルUI（Canvas API）でエンジンの挙動を視覚的に観察できます。

## 🚀 主な機能と特徴

1. **エンジン慣性（応答遅延）の再現**
   - スロットルレバーの動きに対して、エンジン回転数（CORE RPM）や推力が一瞬で変わるのではなく、一次遅れ系（物理的な慣性）を挟んで滑らかに追従します。
2. **ミリタリー推力とアフターバーナー（A/B）の動的切り替え**
   - スロットル 0% 〜 70%：ミリタリー領域（通常最大出力まで）
   - スロットル 71% 〜 100%：アフターバーナー領域（燃料を排気管内で再燃焼させ、爆発的な推力を得る領域）
3. **高リアルな環境物理モデル**
   - **高度（標準大気モデル）**: 高度が上がるほど空気密度が低下し、吸入空気量が減るため推力が目減りします。
   - **速度（ラム圧効果）**: マッハ数が上昇すると、インテークからの空気の押し込み効果（動圧）により、高マッハ時に推力が一時的に向上します。
4. **マルチカラー・マルチファンクション・ディスプレイ（MFD）**
   - レンダリングに HTML5 Canvas API を使用。
   - アナログ風の円形 RPM ゲージ、縦型推力タコメーター、過去100ステップの推力リアルタイム履歴グラフを搭載。
   - アフターバーナー作動時は、UIとグラフの描画色が安全を示す「グリーン/シアン」から、危険・超高出力を示す「レッド/オレンジ」へとトランスフォームします。

## 🛠️ 技術スタック

- **ランタイム**: Node.js
- **GUI フレームワーク**: Electron
- **グラフィック**: HTML5 Canvas API (Pure JavaScript / 外部ライブラリ非依存)
- **デザイン**: CSS3 (Cyberpunk / Military HUD スタイル)
